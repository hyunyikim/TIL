### 트랜지션

- 일정 기간에 걸쳐 속성의 변화가 일어날 수 있음
- 속성 변경이 즉시 영향을 미친다
- 이벤트 트리거에 의해 시작된다(click, hover ...)

### 애니메이션

- 키프레임으로 애니메이션을 나타냄
- 시작과 끝, 중간 단게의 애니메이션을 제어할 수 있음
- 이벤트 트리거가 필요 없다
