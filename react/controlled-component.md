### Controlled Component

- 제어 컴포넌트
- 리액트 state로 값이 관리되는 폼 컴포넌트
- state 변경시 이벤트 핸들러로 값 업데이트

### Uncontrolled Component

- 비제어 컴포넌트
- DOM으로 값이 관리되는 컴포넌트
- ref로 DOM의 입력값을 가져옴
- 파일 입력 태그는 항상 비제어 컴포넌트이다, ref로 값을 가져와야 함
